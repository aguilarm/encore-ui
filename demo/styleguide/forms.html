Design Patterns within Encore

  • Encore uses forms both in separate pages - see our example, which is used when you create a new object such as a Cloud Server or a Database under Encore Cloud, or within a modal such as when a particular action needs to be performed in a server detail page.
  • At this time, Encore only uses stacked forms. This may change later.
  • See the color section for details about how we style and color buttons, but in a nutshell: submit buttons should use the green submit class unless you're deleting or formatting, which should be the negative class. If you need to use a button in your field for auxillary purposes, make it blue.
